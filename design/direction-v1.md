| **PRINT** | `#FF6B6B` Riso 코럴 | `#1E2A52` Riso 미드나잇 | `#FFD93D` Riso 옐로 | 인쇄소 견본 |
| **DAY** | `#FFFF00` 형광 노랑 | `#000000` 순흑 | `#FF0080` 핫 핑크 | 거리 광고지, 마음스튜디오 |

- **검정과 흰색은 색이다.** 무채색 ≠ 안전한 디폴트
- COLOR 버튼은 5개 순환 (4→5)

## 2. 그래픽 — 자모 골격(子母 骨格)

GT Mechanik 5종(구체/폴카닷/격자/동심원/파동)을 *전부 폐기*. 새 어휘:

| ID | 이름 | 형태 | 의미 |
|---|---|---|---|
| `ㅇ` | 원음 | 거대한 단일 원 (외곽선만, 미세 회전) | 발화의 시작 |
| `ㄱ` | 모서리 | 거대한 ㄱ자 코너 (직각/대각) | 굴절, 외벽의 모서리 |
| `ㅅ` | 산형 | 거대한 ㅅ/V (peak) | 솟아오름, 11개 메시지의 떠오름 |
| `─` | 단선 | 굵은 가로선 1–3개 (오프셋 미세 진동) | 발화의 행, 외벽 그리드 |
| `※` | 별표 | 한국식 별표 6획 (방사) | 강조, 별자리 |

특징:
- **모두 *형태 자체*가 강함.** 패턴 아닌 *형태*.
- **굵기 일관:** 모두 두꺼움 (stroke 12–20px @ 380x260 viewBox)
- **모션은 절제:** 회전/맥동/오프셋 — *부드럽지 않게*

## 3. 광학 혼합 — Karel Martens

그래픽은 더 이상 단색 불투명이 아님:
- `fill-opacity: 0.7` (반투명)
- 컨테이너에 `mix-blend-mode: multiply` (밝은 bg) 또는 `screen` (어두운 bg)
- 텍스트가 그래픽 위/아래로 겹칠 때 *세 번째 색*이 광학적으로 생성

STAR 버튼 동작 변경:
- 단순 "5개 중 하나 픽" → **모듈러 구성**: 2개를 동시에 배치 (다른 위치/크기/회전) → 충돌과 혼합

## 4. Riso 물성

화면 전체에 물성 추가:
- **그레인 노이즈 오버레이** — SVG `feTurbulence` 필터로 미세 입자, opacity 0.08–0.15
- **레지스트레이션 오프셋** — 텍스트의 색 카피 두 장 1.5px 어긋남 (graphic 색 + text 색)
- **약간 거친 가장자리** — 굵은 stroke에 `filter: url(#riso-rough)` 적용 가능

## 5. Z축 깊이

지금까지 2D. v1부터 Z축:

| 상황 | 변환 |
|---|---|
| 줄 펄스 활성 줄 | `translateZ(20px)` (앞으로 살짝) + 미세 색 강조 |
| 비활성 줄 | `translateZ(-10px)` (뒤로) |
| Phase 06 도킹 | 메시지가 `rotateX(-65deg) translateZ(-600px) scale(0.4)` 으로 외벽 깊이로 사라짐 + 잔상 에코 3장 |
| 글리프 호버/터치 | 손가락 위치 기준 `rotateY(±8deg)` parallax |

부모 컨테이너: `perspective: 1200px`.

## 6. 컨트롤 종속

여전히 알약이지만 *더 작게*:
- 폰트 크기 9px → 8px
- 패딩 6→4px
- 비활성 옵션은 opacity 0.4
- 라벨이 정보-시각화 같음: `WGHT  ●○○○` (4단계 dot indicator)

**진짜 종속:** 다음 라운드에서 알약 폐기 → 손가락 드래그로 직접 변형 (글리프 위에서 좌/우 = 굵기, 상/하 = 폭). 이번엔 안 함 (스코프 관리).

## 7. 모션 — 절제하되 강하게

- **부드러운 ease 거부.** `cubic-bezier(0.2, 0, 0, 1)` (강한 출발, 강한 도착) 사용
- **즉시성:** 색·팔레트 전환은 100–200ms (현재 500ms는 *지나치게 평온*)
- **잔상:** 활성 → 비활성 전환 시 *짧은 trail* (200ms opacity 잔상)

## 8. 안티-패턴 (v0보다 더 강하게)

- 🚫 alpha 1.0 그래픽 — 반드시 혼합 가능해야 함
