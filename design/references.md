## R01 · Powers of Ten 리소 타이포 포스터

- **출처:** Instagram @ DI35rBzC3ZJ — Eco Kino 영화 상영 포스터
- **사용자 메모:** 리소로 인쇄한 타이포 포스터. "정말 강조"가 되면 좋겠다. 모션이나 효과로 들어갈 수 있을 듯.

**조형 신호:**
- 반복/메아리/에코 = 강조 자체. 텍스트가 계단형으로 자기 증식
- 타입 = 이미지. 글자가 화면을 점령 (데코 아님)
- 리소 인쇄 물성 — 색 어긋남, 거친 텍스처, 두색 인쇄 대비
- 부드러운 페이드 아닌 "쾅쾅쾅" 박자감

**DOCK 적용 가능 지점:**
- 줄 펄스 모션을 부드러운 굵기 변화 → 잔상 에코로 전환 가능
- Phase 06 도킹 클라이맥스에 텍스트 증식 모션
- 색 — 단색 + 강한 대비 1색 방향 검토 (현 10팔레트는 너무 화려할 가능성)
- SIZE 한도 (현 72px) 재고 — 더 크게 갈 여지

## R02 · 워드마크 비틀림 모션

- **출처:** Instagram @ DXVvVw7Dc-- (img 2)
- **사용자 메모 (인용):** "워드마크의 비틀린 형태를 통해 역동적인 인상을 전달했습니다. 이 요소는 모션에서 더 적극적으로 활용되어 원근감을 만들고, 공간감과 움직임을 강조합니다. 빠른 속도에서 느껴질 수 있는 부담감은 줄이면서도, 역동적인 에너지는 효과적으로 살렸습니다."
- **요지:** 단어들이 역동성을 가지길 원함

**조형 신호:**
- 워드마크 = 폼 + 모션이 분리 불가
- 평면 아닌 깊이/원근/공간감
- 절제된 역동 (글리치/폭발 아님)

**R01 vs R02 긴장 (중요):**

| | R01 Powers of Ten | R02 워드마크 모션 |
|---|---|---|
| 시간성 | 정지된 인쇄 | 움직임이 본질 |
| 형태 | 반복/스탬프 | 변형/휘어짐 |
| 공간감 | 평면 위 누적 | 깊이/원근 |
| 메타포 | 아날로그 인쇄 | 디지털/3D |

**둘 다 다른 시점에서 활용 가능:**
- 정지/대기 상태 → R01 (메시지가 외벽에 자리잡음)
- 등장/변화 상태 → R02 (메시지가 외벽에 도착, 풍경에 합류)

**DOCK 적용 가능 지점:**
- 줄 펄스를 굵기 변화만 아니라 *형태 변화*까지 — 활성 줄이 살짝 기울거나 평체로 펴짐
- Phase 04→05 전환: 글자가 비틀리며 이동
- Phase 05→06 (맡기기): 메시지가 평면을 떠나 외벽 깊이로 들어가는 원근 회전

## R03 · Grilli Type — 단일 글자 type tester (a)

- **출처:** Instagram @ DV_O4ZsDtIE (img 2) — Mechanik 제작사 Grilli Type의 다른 작업
- **사용자 메모:** 우리 현재 흐름(쓰기 → 컨트롤 → 완성)이 *상투적*. Grilli는 한 글자에만 먼저 집중해서 자형을 확정. 우리도 흐름 바꿔볼 만함.

**조형 신호 (이미지):**
- 검정 배경 + 거대한 단일 글자 (light grey on black)
- 옵티컬 사이즈/폭/굵기 축이 위에 작게 다이어그램으로
- 글자가 주인공, 컨트롤이 조연 (현 우리 비율과 반대)
- 검정 BG → 사용자의 이전 직감 B를 Grilli 본인들이 정통화

**흐름 재구성 옵션 (별도 메모 design/flow-options.md 참조):**
- Option X (급진): 한 자로 톤 결정 → 그 톤으로 글 쓰기
- Option Y (보수적): 글 쓰기 → 한 자로 톤 발견 → 전체 미세조정
- Option Z (통합): 마지막 친 글자가 라이브 글리프 테스터. 쓰기+조형 동시.

**위험 신호:** DOCK 명세서가 "Phase 4 작성 → Phase 5 톤 결정" 순서를 *원칙*으로 못박았다면 Option X 충돌. 재확인 필요.

## R04 · Karel Martens + 마음스튜디오 — 색 언어

- **출처:** Karel Martens "Still Moving" 전시 포스터 (mohs.official 인스타) + maumstudio.co.kr 작업들
- **사용자 메모:** 색감을 좋아함. 컬러 언어로 가져가고 싶음.

**조형 신호:**
- 순색 자신감 — 파스텔/코퍼레이트 거부. 적/청/노/시안/핑크/녹 풀 채도
- 투명도 + 겹침 = 색 혼합 (Karel Martens 시그니처: 두 색 겹치면 광학적으로 세 번째 색 생성)
- 적은 수 단단하게 (3–4색)
- 한국 모더니즘 형광 어휘 — 형광 노랑/핫 핑크/검정 앵커

**R03과의 긴장 + 화해:**
- R03 검정 — 명상/의례 (글자 발견 순간)
- R04 다채 — 합창/활기 (메시지가 외벽 풍경에 자리잡은 상태)
- *시점이 다를 뿐* — 모순 아님

**DOCK 적용 가능 지점:**
1. 팔레트 수 감축 + 의도 부여 — 10개 랜덤 → 3–4개 큐레이팅된 무드:
   - Night (검정 + 1강조 + 흰글자) ← R03
   - Print (Riso 듀오톤) ← R01
   - Day (형광 노랑/핫 핑크/검정 트라이어드) ← R04/Maum
   - Classic (GT Mechanik 핑크, 디폴트 유지)
2. 그래픽-색 혼합 — 그래픽 SVG 반투명화로 텍스트와 광학적 혼합
3. STAR 버튼의 의미 확장 — 그래픽뿐 아니라 형태+색+위치 무작위 충돌하는 모듈러 시스템
